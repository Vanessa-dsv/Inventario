🛠 Como configurar (primeira vez)
1. Crie a planilha Google Sheets
Abra o Google Drive → Novo → Planilhas. Copie o ID da URL:
docs.google.com/spreadsheets/d/[ID AQUI]/edit

2. Crie o Apps Script
  1. Acesse script.google.com → Novo projeto
  2. Apague o conteúdo e cole o código do arquivo codigo-apps-script.gs
  3. Substitua COLE_AQUI_O_ID_DA_PLANILHA pelo ID copiado
  4. Salve (Ctrl+S)

3. Publique como Web App
  1. Clique em Implantar → Nova implantação
  2. Tipo: App da Web
  3. Executar como: Eu (sua conta)
  4. Quem pode acessar: Qualquer pessoa
  5. Clique em Implantar e autorize
  6. Copie a URL do Web App

4. Configure esta página
Cole a URL do Web App no campo acima e clique em Salvar e conectar.
